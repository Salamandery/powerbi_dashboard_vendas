## 📊 Sobre o Projeto
Dashboard interativo de vendas desenvolvido no **Power BI**, utilizando dados reais de vendas, clientes, produtos e lojas. O objetivo é fornecer insights estratégicos para tomada de decisão, com visualizações dinâmicas, KPIs, segmentações e análises temporais.

---

## 🛠️ Tecnologias e Bibliotecas
- **Power BI Desktop**: Criação e visualização do dashboard.
- **DAX (Data Analysis Expressions)**: Medidas, KPIs e cálculos avançados.
- **Power Query**: ETL para tratamento e modelagem dos dados.
- **Excel**: Fontes de dados em `.xlsx`.

---

## 📐 Padrões de Projeto
- **Modelagem Estrela**: Separação entre tabelas fato (vendas) e dimensões (produtos, clientes, lojas, datas).
- **ETL Automatizado**: Power Query para padronização e atualização dos dados.
- **Medidas DAX**: KPIs centralizados e reutilizáveis.
- **Layout Responsivo**: Visual limpo, intuitivo e adaptado para diferentes resoluções.

---

## ⚙️ Setup e Configuração
1. **Pré-requisitos:**
   - [Power BI Desktop](https://powerbi.microsoft.com/pt-br/desktop/) instalado.
   - Microsoft Excel (opcional, para edição das bases).
2. **Clone o repositório:**
   ```bash
   git clone https://github.com/Salamandery/powerbi_dashboard_vendas.git
   ```
3. **Abra o arquivo** `Painel Vendas.pbix` no Power BI Desktop.
4. **Atualize as conexões** caso os arquivos `.xlsx` estejam em outro diretório:
   - Vá em **Transformar Dados > Gerenciar Fontes** e ajuste os caminhos dos arquivos.

---

## 💡 Como Usar
- Navegue entre as páginas do dashboard para visualizar:
  - Faturamento, quantidade vendida, ticket médio, produtos e lojas de destaque.
  - Filtros por período, loja, produto e cliente.
  - KPIs e gráficos dinâmicos.
- Para modificar ou atualizar os dados, substitua os arquivos `.xlsx` pelas novas versões e atualize no Power BI.
